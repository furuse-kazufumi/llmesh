# LLMesh 論文素材集

本ディレクトリは **公益社団法人 精密工学会（JSPE）** へ投稿予定の 4 本の論文の
素材ファイル群です。各論文は LLMesh の画像処理機能を共通基盤とし、
独立したテーマで切り出しています。

## 精密工学会向けの統一スタイル

- **言語**: 和文または英文（精密工学会誌は両対応）
- **長さ**: 査読論文 8 ページ以内（A4・2 段組）/ 学術講演会 4 ページ以内
- **図表**: モノクロ印刷を想定したコントラスト
- **参考文献**: JSPE スタイル（番号付き、著者名・誌名・巻号・ページ・西暦）
- **キーワード**: 各論文に 5–6 個

## 共通キーワード（4 論文で共通）
- ローカル LLM（Local Large Language Model）
- 産業 IoT / IIoT
- プライバシー保護
- 自動外観検査（AOI）

## 論文ごとの追加キーワード
- P1: 3D センサー要約, 深度カメラ, イベントカメラ
- P2: プライバシーパイプライン, PII 検出, 画像ファイアウォール
- P3: 自動外観検査, MT 法, 根本原因分析（RCA）
- P4: イベントカメラ, 高速計測, 振動解析
