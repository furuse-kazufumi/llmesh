## 設計メモ

### MTEngine

- MT法（マハラノビス-タグチ法）の核心: 正常データの相関行列逆行列でスケールした距離
- `MD = sqrt(z^T R^{-1} z / p)` — p は特徴量数（正規化スケーリング）
- 零分散特徴量は std=1.0 に置き換えて ZeroDivisionError 回避
- 特異相関行列は `scipy.linalg.pinv()` で擬似逆行列にフォールバック
- numpy/scipy 未インストール時は `RuntimeError` で明確に失敗（graceful degradation）

### XbarRChart

- ASTM / ISO 8258 準拠の A2/D3/D4 係数テーブル（n=2〜10）
- UCL_x = x̄̄ + A2·R̄、LCL_x = x̄̄ - A2·R̄
- UCL_R = D4·R̄、LCL_R = D3·R̄（n≤6 は D3=0 で下限なし）
- 純 stdlib 実装（math, statistics のみ）

### CUSUMChart

- 二方向 CUSUM（Page's scheme）
  - S+ = max(0, S+ + (x - μ0 - k))  ← 上方シフト
  - S- = max(0, S- + (μ0 - x - k))  ← 下方シフト
- sigma 指定時は z-スコア変換後に判定（σ 未知の場合は raw 値でそのまま）
- バグ修正: 実装時に S- の式を誤記（`-(x-target-k)` → `+(target-x-k)` に修正）

## CLI フロー

```
# 1. 正常データ収集（stdin から）
echo -e "2.1 2.3 2.0\n1.9 2.2 2.1\n2.0 2.1 2.2" | \
  llmesh mt-collect --device smt_01 --duration 0 --output normal_data.npz

# 2. ユニットスペース学習
llmesh mt-train --input normal_data.npz --device smt_01

# 3. リアルタイム推論
echo "5.0 5.0 5.0" | llmesh mt-infer --model unit_space_smt_01.npz --threshold 3.0
```
